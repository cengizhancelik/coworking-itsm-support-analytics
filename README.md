# Coworking Operasyonlarında ITSM Destek Talebi Analitiği ve Operasyonek Süreç Verimliliğinin Geliştirilmesi
Bu repository, İstanbul Medipol Üniversitesi Yönetim Bilişim Sistemleri Bölümü kapsamında hazırlanan bitirme projesini içermektedir.

## Proje Konusu

Bu çalışmada coworking alanlarında oluşan destek talepleri;

- İnternet ve teknik altyapı
- Erişim kontrolü
- Bakım ve onarım
- Üyelik işlemleri
- Operasyonel destek süreçleri

gibi kategoriler üzerinden incelenmiştir.

## Projenin Amacı

Çalışmanın amacı, ITSM destek taleplerini veri analitiği yöntemleriyle inceleyerek tekrar eden problemleri, hizmet performansını ve süreç iyileştirme fırsatlarını belirlemektir.

## Kullanılan Araçlar

- Microsoft Word
- Microsoft Excel
- Power BI
- Python
